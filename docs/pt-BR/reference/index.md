# Referência

As páginas de referência reúnem detalhes factuais que você pode querer consultar rapidamente ao trabalhar com o Koharu.

## Referências disponíveis

- [Referência da CLI](cli.md): flags de inicialização, comportamento do servidor local e padrões comuns de runtime
- [Referência da API HTTP](http-api.md): endpoints REST locais, nomes de eventos de stream, payloads e ordem do workflow
- [Referência das ferramentas MCP](mcp-tools.md): nomes das ferramentas MCP embutidas, parâmetros e fluxo de uso sugerido
- [Referência de configurações](settings.md): aparência, seleção de engine, chaves de provedor, configuração de runtime e comportamento da página About
- [Atalhos de teclado](keyboard-shortcuts.md): os atalhos padrão do editor atualmente documentados na UI

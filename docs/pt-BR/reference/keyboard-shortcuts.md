# Atalhos de teclado

Estes são os controles padrão do editor atualmente documentados no app.

| Atalho | Ação |
| --- | --- |
| `Ctrl` + roda do mouse | Aumentar ou diminuir o zoom |
| `Ctrl` + arrastar | Deslocar a tela |
| `Del` | Excluir o bloco de texto selecionado |
